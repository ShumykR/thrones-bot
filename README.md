# ⚔️ ThroneChat Bot

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)
![aiogram Version](https://img.shields.io/badge/aiogram-3.x-success)
![SQLAlchemy Version](https://img.shields.io/badge/SQLAlchemy-2.0-red)
![License](https://img.shields.io/badge/license-MIT-green)

**ThroneChat** — це багатофункціональний ігровий Telegram-бот для групових чатів, натхненний всесвітом «Гри Престолів». Він перетворює звичайний чат на поле політичних інтриг, боротьби за владу та військових завоювань. Гравці збирають армії, захоплюють замки Вестеросу, стають васалами або ж повстають проти своїх сюзеренів, щоб врешті-решт захопити Залізний Трон.

---

## 🌟 Ключові можливості

👑 **Система Короля**
Гравець, який захопив столицю (Висад), отримує титул Короля. Король має унікальні адміністративні права в групі: може відправляти неугодних у "Темницю" (мут), конфісковувати армії та видавати королівські укази (закріплені повідомлення).

🏰 **Інтерактивний WebApp (Mini App)**
Бот має вбудований фронтенд-додаток з картою Вестеросу. Гравці можуть:
- Переглядати розташування замків на красивій карті.
- Бачити власників та гарнізони у реальному часі.
- Відправляти армію на штурм за допомогою зручного повзунка.

⚔️ **Військова та Економічна системи**
- **Армія:** Щоденно (та щогодинно) лорди отримують нових воїнів. Чим більше замків у власності — тим більший приріст та ліміт армії.
- **Штурм (`/attack`):** Бої розраховуються асинхронно з рандомізацією (±20% до сили).
- **Гарнізон:** Захисники мають бонуси при обороні стін замку.

🤝 **Дипломатія, Васали та Змова**
- **Маріонетки (Васали):** Гравці, які втратили всі замки і програли битву, стають маріонетками переможця, сплачуючи йому податки.
- **Індепенденс-поїнти (IP):** За активність у чаті маріонетки накопичують бали, які дозволяють влаштувати бунт (`/rebel`) або саботаж (`/sabotage`).
- **Змова (`/conspiracy`):** Механіка таємного голосування через Deep Linking. Гравці можуть об'єднатися для повалення Короля без його відома.

## 🛠 Технологічний стек

* **Бекенд:** Python 3.10+, [aiogram 3.x](https://docs.aiogram.dev/en/latest/) (асинхронний фреймворк для Telegram).
* **База даних:** SQLite (через `aiosqlite`) + [SQLAlchemy 2.0](https://www.sqlalchemy.org/) (ORM).
* **API та WebApp Сервер:** `aiohttp` (працює паралельно з ботом в одному Event Loop).
* **Планувальник завдань:** `APScheduler` (для щогодинного нарахування військ та івентів).
* **Фронтенд (Mini App):** Vanilla JavaScript, HTML, CSS (динамічний рендер SVG/абсолютних координат поверх зображення карти).

## 🚀 Встановлення та запуск

1. **Клонування репозиторію:**
   ```bash
   git clone https://github.com/yourusername/ThroneChat.git
   cd ThroneChat
   ```

2. **Створення та активація віртуального середовища:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   .venv\Scripts\activate     # Для Windows
   ```

3. **Встановлення залежностей:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Налаштування середовища:**
   Скопіюйте файл `.env.example` у `.env` та заповніть свої дані:
   ```bash
   cp .env.example .env
   ```
   *Обов'язково вкажіть `BOT_TOKEN` від @BotFather, `BOT_USERNAME` та `CHAT_ID` вашої групи.*

5. **Запуск бота:**
   ```bash
   python -m bot.main
   ```
   *При першому запуску бот автоматично створить базу даних `thronebot.db` і заповнить її замками.*

## ⚙️ Баланс та Конфігурація

Увесь ігровий баланс винесено у файл `config/config.py`. Ви можете легко змінювати гру під стиль вашого чату без втручання в основний код:
- `MIN_ATTACK_ARMY` — мінімальна кількість військ для атаки.
- `BATTLE_RESOLVE_MINUTES` — час очікування результатів битви.
- `PUPPET_TAX_RATE` — відсоток податку, який сплачують васали.
- `CONSPIRACY_DURATION_H` — тривалість таємного голосування за скинення Короля.
... та багато іншого.
